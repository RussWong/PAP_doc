PAP

本项目基于链家二手房数据集，经过数据处理、特征工程、模型训练及预测等环节，对二手房成交价完成预测，并且对预测结果进行解释和分析。

运行方法

  • python -m luigi --module main Preprocessing --local-scheduler - 数据预处理，输出数据预处理后的数据集

  • python -m luigi --module main Anomaly --local-scheduler - 异常检测，输出异常检测后的数据集

  • python -m luigi --module main Encoding --local-scheduler - 特征编码，输出target encoding后的数据集

  • python -m luigi --module main FeatureSelection --name-of-feature 15 --local-scheduler - 特征选择，输出特征选择后还剩15个最重要特征的数据集

  • python -m luigi --module main BRreg/XGBreg/Lreg/Breg/GBreg --name-of-model 模型名称 --local-scheduler- 训练模型，模型名称可选bayesianridge,xgboost,lasso,bagging,gradientboosting

  • python -m luigi --module main Analysis --name-of-model 模型名称 --local-scheduler - 模型分析，输出该模型中特征对预测结果的贡献图，模型名称可选bayesianridge,xgboost,lasso,bagging,gradientboosting
